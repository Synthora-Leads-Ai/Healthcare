## 🎨 Color Scheme

### Primary Colors
- **Medical Blue**: #0284c7 (Buttons, links, accents)
- **Medical Blue Hover**: #0369a1
- **Light Blue**: #f0f9ff (Backgrounds)
- **Cyan**: #0891b2 (Secondary accents)

### Neutral Colors
- **White**: #ffffff (Backgrounds)
- **Gray 900**: #111827 (Text)
- **Gray 700**: #374151 (Secondary text)
- **Gray 100**: #f3f4f6 (Light backgrounds)

### Accent Colors
- **Green**: #10b981 (WhatsApp, success)
- **Yellow**: #fbbf24 (Stars, ratings)
- **Red**: #ef4444 (Emergency, important)

## 📱 Mobile View

All sections stack vertically on mobile with:
- Hamburger menu for navigation
- Full-width cards
- Touch-friendly buttons (min 44px)
- Optimized images
- Readable text (16px minimum)

---

## ✨ Animations

- **Fade In**: Sections fade in as you scroll
- **Slide In**: Content slides from left/right
- **Hover Effects**: Cards lift on hover
- **Pulse**: Call button pulses to draw attention
- **Smooth Scroll**: Smooth scrolling between sections

---

## 🎯 Key Visual Elements

1. **Doctor Badge**: Circular badge with initials "DS"
2. **Stats Cards**: 4 cards showing key metrics
3. **Service Icons**: Lucide React icons for each service
4. **Rating Stars**: Yellow stars for testimonials
5. **Gradient Backgrounds**: Subtle blue gradients
6. **Shadow Effects**: Elevated cards with shadows
7. **Rounded Corners**: Consistent 0.75rem radius
8. **Floating Buttons**: Fixed position, always visible
