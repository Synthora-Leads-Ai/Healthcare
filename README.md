# Soni Physiotherapy Centre & Home Healthcare Services

A modern, responsive website for Soni Physiotherapy Centre & Home Healthcare Services in Jabalpur, Madhya Pradesh.

## About

Soni Physiotherapy Centre is a doctor-led healthcare facility founded by Dr. Arvind Soni (MPT - Ortho) with over 10 years of experience in providing quality physiotherapy and home healthcare services.

## Features

- **Responsive Design**: Fully responsive across all devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Comprehensive Services**: Detailed information about all physiotherapy and home healthcare services
- **Easy Contact**: Multiple contact options including WhatsApp integration
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Fast Performance**: Built with Vite for optimal loading speed

## Tech Stack

- **React 19** - UI Framework
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **React Router** - Navigation
- **Lucide React** - Icons

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd soni-physiotherapy
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

## Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.

## Preview Production Build

```bash
npm run preview
```
