# Soni Physiotherapy Centre & Home Healthcare Services

Modern, responsive website for Soni Physiotherapy Centre in Jabalpur, Madhya Pradesh.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Visit: `http://localhost:5173`

## 📦 Build

```bash
npm run build
npm run preview
```

## 🎯 Features

- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Online appointment booking
- ✅ Contact form
- ✅ WhatsApp integration
- ✅ Service showcase with images
- ✅ Gallery page
- ✅ Google Maps integration
- ✅ SEO optimized

## 🛠️ Tech Stack

- React 19
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- Lucide React

## 📝 Customization

Edit `src/config/clinicInfo.js` to update:
- Contact information
- Services
- Testimonials
- Doctor details
